# Telegram bot

## Описание проекта
  Телеграм бот, состоящий из 15 вопросов по языку Python.
  По ходу прохождения квиза, бот оповещает, правильно ли пользователь
  ответил на вопрос.
  Если квиз завершен раньше, бот выводит текущую статистику.
  Так же, при полном прохождении квиза, бот выводит количество верных и неверных ответов.
